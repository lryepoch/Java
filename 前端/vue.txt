·基本工具安装：
	··安装node.js安装包；
	··(去到项目目录下)安装npm的阿里镜像，原生npm下载依赖包过慢，镜像可以提高下载效率：npm i -g cnpm --registry=https://registry.npm.taobao.org
	··安装Vue的脚手架工具vue-cli：npm i -g vue-cli
	··测试是否安装成功：vue -V    //注意：-V是大写的V，不是小写的v
	··安装webpack工具（如果以前没有安装webpack，一定要安装！！！）：cnpm install -g webpack

vue init webpack 项目名：创建项目
npm install:安装依赖（会生成node_modules文件夹）
npm run dev：启动运行vue项目
npm run build：打包上线，生成dist文件夹（包含static文件夹和index.html文件）
